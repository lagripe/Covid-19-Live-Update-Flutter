# Getting started

1. [Setup Flutter](https://flutter.dev/docs/get-started/install)

2. Clone the repo

    ```
    $ git clone https://github.com/lagripe/Covid-19-Live-Update-Flutter
    $ cd Covid-19-Live-Update-Flutter/covid
    ```
3. Run
        `$ flutter run`
